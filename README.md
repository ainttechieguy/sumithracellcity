# Sumithra Cell City

Welcome to the official frontend repository for **Sumithra Cell City** - Chennai's trusted smartphone experts. 
Established in 2005, Sumithra Cell City specializes in smart mobile services, including expert care for all Apple products and major smartphone brands.

## 🚀 Tech Stack

This project is built using modern web technologies to ensure a lightning-fast, highly interactive, and SEO-optimized user experience:

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM (v6)
- **Animations**: Motion (Framer Motion)
- **Forms**: React Hook Form with Zod validation
- **SEO**: React Helmet Async
- **Icons**: Lucide React & SimpleIcons

## ✨ Key Features

- **Interactive 3D Hero**: Features a custom-built 3D animated smartphone with floating, exploded internal parts responding to mouse movements.
- **Multi-Page Architecture**: Fully routed multi-page layout (Home, About, Services, Brands, Process, Reviews).
- **Responsive Design**: Tailored experiences across mobile, tablet, and desktop devices.
- **Dark Mode**: Complete system-aware dark mode support with tailored glows, shadows, and glassmorphism.

## 📦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
   ```sh
   git clone git@github.com:ainttechieguy/sumithracellcity.git
   ```
2. Navigate into the frontend directory
   ```sh
   cd sumithracellcity/frontend
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

## 🛠 Usage

To start the local development server:

```sh
npm run dev
```

Your app will be available at `http://localhost:5173/` (or whichever port Vite provides).

To build the project for production:

```sh
npm run build
```
